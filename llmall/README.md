# LLM All Prompt Relay

Chrome のツールバーアイコンを押すとポップアップが開き、入力した質問を `Gemini` / `ChatGPT` / `Claude` に同時送信する拡張です。

## 使い方

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」からこのディレクトリを選ぶ
4. 拡張アイコンをクリックするか、`Ctrl+Shift+P` / `Command+Shift+P` でポップアップを開く
5. 質問を入力する
6. 必要なら `送信せず開く` をオンにする
7. `Ctrl+Enter` または `Cmd+Enter`、もしくは送信ボタンで実行する
8. 実行するとポップアップはすぐ閉じ、現在の Chrome ウィンドウ内で `Gemini` / `ChatGPT` / `Claude` の新規チャット用タブを3つバックグラウンドで開く

## 動作

- 通常時は各タブに入力したあと、そのまま送信します
- 送信要求を受け付けたらポップアップはすぐ閉じ、送信処理はバックグラウンドで続きます
- 作成したタブは自動で前面に切り替えません
- 送信に失敗したサービスがある場合は Chrome 通知で失敗内容を知らせます
- `送信せず開く` をオンにすると、各タブに入力した状態で止めます
- ポップアップの起動ショートカットは Chrome の `chrome://extensions/shortcuts` から変更できます

## 構成

- `manifest.json`: Manifest V3 の設定
- `popup.html` / `popup.css` / `popup.js`: ポップアップ UI
- `background.js`: 各サービスの新規タブ作成、入力・送信処理

## 注意点

- それぞれのサイトでログイン済みである必要があります
- サイト側の DOM 構造やボタンのラベルが変わると送信できなくなることがあります
- 送信のたびに新しい3タブを作るので、連続で使うとタブ数は増えます
- 新規会話 URL や送信 UI はサービス側変更の影響を受けます
