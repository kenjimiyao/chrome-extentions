# LLM All Prompt Relay

Chrome のツールバーアイコンを押すとポップアップが開き、入力した質問を `Gemini` / `ChatGPT` / `Claude` に同時送信する拡張です。

## 使い方

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」からこのディレクトリを選ぶ
4. 拡張アイコンをクリックして質問を入力する
5. `Ctrl+Enter` または `Cmd+Enter`、もしくは送信ボタンで実行する
6. 送信時に現在の Chrome ウィンドウ内で `Gemini` / `ChatGPT` / `Claude` の新規チャット用タブを3つ作り、それぞれへ送信する

## 構成

- `manifest.json`: Manifest V3 の設定
- `popup.html` / `popup.css` / `popup.js`: ポップアップ UI
- `background.js`: 各サービスの新規タブ作成、入力・送信処理

## 注意点

- それぞれのサイトでログイン済みである必要があります
- サイト側の DOM 構造やボタンのラベルが変わると送信できなくなることがあります
- 送信のたびに新しい3タブを作るので、連続で使うとタブ数は増えます
- 新規会話 URL や送信 UI はサービス側変更の影響を受けます
